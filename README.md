## 文件说明
* `clock.py`: 主程序 (过时的)
* `vpn.py`: 深信服webvpn登录

* `get_session.py`: 在licsber.wisedu.get_session.py的基础上增加webvpn环境下的authserver (过时的，Licsber工具库已更新相关功能)
